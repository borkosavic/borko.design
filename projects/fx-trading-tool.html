FX Trading Tool

An internal tool used by Singapore and Hong Kong branches of HSBC Bank for private bankers to facilitate foreign currency exchange on behalf of high-ranking clients.

Duration 2021 (6 months)
Skills UX Design, UI Design
Project Hero Image
⚠️

Confidentiality Notice

Due to strict confidentiality and security constraints, I'm not permitted to show any visuals from this project. The following describes the design journey and challenges encountered.

Project Overview

An internal HSBC tool for private bankers to execute complex foreign exchange transactions, including currency and bullion trades, for high-net-worth clients.

This tool was specifically designed for the Singapore and Hong Kong branches of HSBC Bank, enabling private bankers to handle sophisticated financial transactions with the highest levels of security and compliance.

My Role

  • • Designed user flows and wireframes for the trading interface
  • • Delivered prototypes that adhered to HSBC's strict UI guidelines
  • • Collaborated with developers to ensure seamless implementation

Key Challenges

  • • Working within strict confidentiality and security constraints
  • • Limited UI components due to pre-approved design systems
  • • No access to real users for testing

Solution

Stakeholder-Driven Design

Used insights from stakeholders to define user needs and workflows, working closely with business analysts and product managers to understand the complex requirements of foreign exchange trading.

Stakeholder Workshop Documentation

HSBC Design System Compliance

Leveraged HSBC's established design system to ensure consistency and compliance with bank's security standards, while creating an intuitive interface for complex financial operations.

HSBC Design System Components

Detailed Documentation

Delivered comprehensive prototypes and design documentation for implementation, ensuring developers had clear guidance for building the secure trading interface.

Design Documentation Package

Key Constraints

Security Requirements

Strict confidentiality and security constraints required careful handling of all design materials and processes.

Design System Limitations

Limited to pre-approved UI components within HSBC's established design system for compliance and consistency.

User Access

No direct access to end users for testing due to security protocols and confidentiality agreements.

Complex Financial Operations

Had to design for highly complex foreign exchange and bullion trading workflows with multiple approval stages.

Results

On-Time Delivery

Successfully delivered the project on time, meeting HSBC's high standards for both usability and security compliance.

Streamlined Workflows

Created efficient workflows for private bankers managing high-value transactions, reducing complexity in critical financial operations.

Security & Compliance

Delivered a solution that met HSBC's rigorous security requirements while providing an intuitive interface for complex financial trading operations, enabling private bankers to serve high-net-worth clients more effectively.
