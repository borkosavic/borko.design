Act2Access

A startup aimed at helping publishers with monetization by providing premium content to users who blocked ads through alternative engagement methods like surveys and A/B testing.

Duration 2017 (2 years)
Skills UX Design, UI Design, Design System
Project Hero Image

Project Overview

A content monetization platform that allows publishers to engage users with ad blockers through alternative methods like surveys and A/B testing, providing valuable information to advertisers while offering premium content access.

Act2Access addressed the growing challenge of ad blockers by creating a win-win solution: users could access premium content without seeing traditional ads, while publishers could still monetize their content through user engagement data.

My Role

  • • Designed UI components and workflows for both users and publishers
  • • Created interactive prototypes for integration into publishing platforms
  • • Conducted user testing to validate MVP features

Key Challenges

  • • Balancing user engagement with revenue generation for publishers
  • • Designing for compatibility across various publishing platforms
  • • Creating compelling alternatives to traditional advertising

Solution

Dual-Interface Design

Designed separate but connected interfaces for end users and publishers, ensuring both audiences could effectively use the platform while maintaining data flow between the two sides.

User & Publisher Interface Flow

Cross-Platform Integration

Created interactive prototypes that could be seamlessly integrated into various publishing platforms, ensuring wide compatibility and easy implementation.

Platform Integration Components

MVP Validation

Conducted extensive user testing with focus groups to validate core features and ensure the value proposition resonated with both users and publishers.

User Testing Sessions

Key Innovation

Ad Blocker Solution

Transformed the ad blocker "problem" into an opportunity by creating alternative engagement methods that provided value to all parties.

Data-Driven Monetization

Enabled publishers to monetize through valuable user insights and engagement data rather than traditional display advertising.

User Choice

Empowered users to choose how they wanted to "pay" for content - through surveys, A/B testing, or other engagement methods.

Advertiser Value

Provided advertisers with high-quality, engaged user data and insights instead of potentially ignored display ads.

Results

Successful MVP Launch

Successfully launched the minimum viable product with positive feedback from both publishers and end users.

Publisher Adoption

Achieved initial publisher adoption with measurable improvements in user engagement and revenue generation.

Actionable Insights

Provided publishers with valuable, actionable insights about their audience while maintaining user satisfaction.

Scalable Framework

Established a solid foundation for continued platform growth and innovation in content monetization.
