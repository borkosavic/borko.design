Rho - Expense Rules

A fintech platform designed to help businesses manage corporate expenses with intelligent rule-based automation and real-time spending controls.

Duration 2024 (8 months)
Skills UX Design, UI Design, Design System
URL rho.com
Project Hero Image

Project Overview

Rho is a business banking and expense management platform that helps companies control spending through intelligent automation. The expense rules feature allows businesses to set up sophisticated spending controls and approval workflows.

This project focused on creating an intuitive interface for configuring complex business rules, making it easy for finance teams to implement and manage spending policies across their organization.

My Role

  • • Designed the complete expense rules configuration interface
  • • Created user flows for rule creation and management
  • • Developed design system components for complex form inputs
  • • Collaborated with product and engineering teams on implementation

Key Challenges

  • • Making complex business logic accessible to non-technical users
  • • Designing for multiple user roles with different permission levels
  • • Ensuring rule conflicts and dependencies are clearly communicated
  • • Creating scalable patterns for future rule types

Solution

Rule Builder Interface

Created an intuitive rule builder that breaks down complex business logic into simple, configurable components. Users can set conditions, actions, and exceptions through a guided interface.

Rule Builder Interface

Visual Rule Management

Designed a comprehensive dashboard for managing all expense rules, with clear visual indicators for rule status, conflicts, and performance metrics.

Rule Management Dashboard

Smart Conflict Detection

Implemented intelligent conflict detection that identifies rule overlaps and provides clear guidance on resolving conflicts before they impact business operations.

Conflict Detection System

Key Features

Flexible Rule Creation

Intuitive interface for creating complex spending rules with conditions, actions, and exceptions tailored to business needs.

Real-time Validation

Immediate feedback on rule configuration with clear error messages and suggestions for improvement.

Role-based Permissions

Different interface views and capabilities based on user roles, from basic users to finance administrators.

Performance Analytics

Comprehensive reporting on rule effectiveness, usage patterns, and impact on business spending.

Results

Improved User Adoption

Simplified rule creation process led to 40% increase in feature adoption among finance teams.

Reduced Configuration Errors

Smart validation and conflict detection reduced rule configuration errors by 60%.

Enhanced Business Control

Businesses reported better expense control and compliance with automated rule enforcement.

Scalable Design System

Created reusable components and patterns that accelerated development of future rule types.
