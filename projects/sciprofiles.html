SciProfiles

A social network for researchers, scientists, and academics designed to help users create profiles, share research, and discover collaborators.

Duration 2020 (1 year)
Skills UX Design, UI Design, Design System
URL sciprofiles.com
Project Hero Image

Project Overview

A networking and collaboration platform for the scientific community, designed to serve diverse user needs like casual networking, research paper sharing, and peer review management.

SciProfiles has a highly diverse user base, encompassing seven distinct personas, each with unique ways of using the platform. Regular users focused on casual networking, some authors searched for co-authors, others were searching for research papers they could cite, and editors sought out peer reviewers.

My Role

  • • Redesigned the information architecture to accommodate variety of user paths
  • • Conducted user research and testing to refine platform features
  • • Developed a Figma-based design system to unify UI development

Key Challenges

  • • Managing needs of seven distinct personas with varying behaviors
  • • Balancing dense scientific data presentation with usability
  • • Ensuring legibility across diverse content types

Solution

Information Architecture Redesign

Streamlined navigation with a card-sorting exercise to restructure information architecture, accommodating the diverse range of user paths and behaviors.

Information Architecture Map

Scalable Design System

Built a robust design system to support scalability and consistency across all platform features, enabling rapid development and maintenance.

Design System Components

User-Centered Testing

Conducted iterative testing to optimize user experience for editors, authors, and researchers, ensuring each persona could efficiently accomplish their goals.

User Testing Sessions

Key Findings

Diverse User Base

Seven distinct personas required unique user paths and interface considerations.

Cultural Differences

Global scientific community brought diverse interaction expectations and preferences.

Information Architecture

Complex scientific data required careful organization and presentation strategies.

Results

20x Growth in MAU

Monthly active users increased dramatically within one year of the redesign launch.

Improved Usability

Enhanced platform usability and satisfaction across all seven user personas.

Company-Wide Design System

The design system was adopted across all MDPI platforms, enhancing development speed and consistency throughout the organization while establishing a foundation for future product development.
