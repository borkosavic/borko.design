What is it about

Researchers around the world were trying to navigate the platform which felt more like a maze than a resource. MDPI is an open-access publisher that prides itself on offering fast and transparent publishing services for peer-reviewed journals across various fields like science, technology, and medicine.

When I joined, the UX/UI was being handled by developers, with occasional input from graphic designers. The goal for the project was to completely revamp SciProfiles - not just to make it more user-friendly but also to demonstrate the value of UX design as a practice.

What did I do?

I was the only designer in a project-based team of developers and managers. I had close collaboration with stakeholders too.

Research

Facilitated all kinds of user research - focus groups, user interviews, card sorting exercises as well as stakeholder interviews.

Design System

SciProfiles was using several UI components from a PHP framework - I completely overhauled it in cooperation with the dev team.

What's the outcome?

The redesign proved to be a great success, achieving a 20x increase in monthly active users - from 5k to 100k.

  • Solid design system and planning proved beneficial to increased speed
  • Expert users don't mind clutter if it increases usability
  • First time getting user testing results I really hadn't anticipated
